# Logos de la galaxie EBOK

Dépose ici les logos des applications EBOK, exactement avec ces noms de
fichier (PNG, **fond transparent de préférence** — les logos sont
affichés en entier, sans rognage ni cadre derrière) :

- `ebokbasketball.png`
- `ebokacademie.png`
- `ebokblog.png`
- `ebokevent.png`
- `ebokmercato.png`
- `eboknotebook.png`
- `ebokplaybook.png`
- `ebokscouting.png`
- `ebokstats.png`
- `ebokvideo.png`

Ils sont affichés automatiquement dans le bandeau qui défile en bas de
la page d'accueil (voir `.galaxy-band` dans `public/css/styles.css` et
`renderGalaxyBand()` dans `public/js/app.js`). Aucune autre action n'est
nécessaire : dès qu'un fichier est déposé sous ce nom, il apparaît sur
le site au prochain chargement.
